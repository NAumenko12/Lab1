# Обработчик строк (C++)

Простая консольная программа на C++ для работы со строками. Программа умеет изменять регистр символов (на верхний и нижний), а также подсчитывать количество гласных букв. Полностью поддерживает как английский алфавит (латиницу), так и русский алфавит (кириллицу). Принимает исходную строку в stdin формате.

## Возможности

- **Перевод в верхний регистр** `toUpperCase(wstring& str)`: Конвертирует все строчные буквы в ЗАГЛАВНЫЕ
- **Перевод в нижний регистр** `toLowerCase(wstring& str)`: Конвертирует все заглавные буквы в строчные
- **Подсчет гласных** `vowelsLength(wstring& str)`: Считает количество гласных букв в введенной строке (поддерживаются гласные русского и английского алфавитов)
- **Поддержка Unicode**: Использование `std::wstring` и широких символов (`wchar_t`) позволяет корректно обрабатывать кириллицу
